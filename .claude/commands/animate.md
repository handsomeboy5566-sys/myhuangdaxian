name: animate
description: Review a feature and enhance it with purposeful animations, micro-interactions, and motion effects.
---

Add purposeful motion to the current feature/page:
- Page load orchestration (staggered reveals)
- State change animations (hover, focus, active)
- Micro-interactions that provide feedback
- Scroll-driven effects (if appropriate)
- Entrance/exit animations for modals/dialogs

**Animation Principles:**
- Use exponential easing (ease-out-quart/quint) for natural deceleration
- Focus on high-impact moments, not scattered micro-interactions
- Use transform and opacity only (don't animate layout properties)
- Avoid bounce/elastic easing (feels dated)
- Respect reduced-motion preferences

Make the interface feel alive and responsive while maintaining performance (60fps).
