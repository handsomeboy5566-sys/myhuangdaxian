---
name: colorize
description: Add strategic color to features that are too monochromatic or lack visual interest.
---

Enhance the current design with strategic color:
- Create a cohesive color palette (dominant + accent colors)
- Use modern CSS color functions (oklch, color-mix)
- Tint neutrals toward brand hue for cohesion
- Ensure accessible contrast ratios
- Support light/dark mode if appropriate

**Color Principles:**
- Dominant colors with sharp accents outperform timid palettes
- Never use pure black (#000) or pure white (#fff)
- Never use gray text on colored backgrounds
- Avoid the "AI color palette": cyan-on-dark, purple-to-blue gradients

Transform boring/monochromatic designs into visually engaging interfaces.
