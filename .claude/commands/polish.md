description: Final quality pass before shipping. Fixes alignment, spacing, consistency, and detail issues that separate good from great.

Run a final quality pass on the current design/code. Fix:
- Alignment issues (things that are 1-2px off)
- Spacing inconsistencies
- Color inconsistencies
- Typography issues (hierarchy, line-height, letter-spacing)
- Missing hover/focus states
- Incomplete responsive behavior
- Visual polish that elevates good to great

Focus on the details that show craftsmanship. Do a thorough review and fix all issues found.
