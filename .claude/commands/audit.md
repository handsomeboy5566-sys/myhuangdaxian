---
name: audit
description: Perform comprehensive audit of interface quality across accessibility, performance, theming, and responsive design.
---

Run a comprehensive quality audit checking:
- **Accessibility**: Color contrast, focus states, semantic HTML, ARIA labels
- **Performance**: Image optimization, bundle size, animation performance
- **Responsive**: Mobile, tablet, desktop breakpoints
- **Theming**: Light/dark mode support
- **Typography**: Hierarchy, readability, font loading

Generate a detailed report with:
- Issues found (categorized by severity: critical, high, medium, low)
- Specific recommendations for each issue
- Prioritized fix list

Then fix all critical and high priority issues automatically.
