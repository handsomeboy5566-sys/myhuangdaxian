name: typeset
description: Improve typography by fixing font choices, hierarchy, sizing, weight consistency, and readability.
---

Fix typography issues in the current design:
- Choose distinctive, beautiful fonts (avoid Inter, Roboto, Arial, system defaults)
- Create clear visual hierarchy with varied weights and sizes
- Implement modular type scale with fluid sizing (clamp)
- Fix line-height, letter-spacing for readability
- Ensure adequate contrast between heading and body text

**Typography Principles:**
- Pair a distinctive display font with a refined body font
- Use a clear type scale (e.g., 1.25 or 1.5 ratio)
- Vary font weights to create hierarchy
- Don't put large icons with rounded corners above every heading
- Make text feel intentional and polished

Transform mediocre typography into a beautiful, readable system.
